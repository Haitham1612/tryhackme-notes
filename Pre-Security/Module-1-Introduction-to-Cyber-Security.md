# Module 1: Introduction to Cyber Security

**Path:** Pre Security (TryHackMe)  
**Completed:** 14 April 2026

## Overview
This module gave a broad introduction to cyber security — covering both the offensive and defensive sides, plus an overview of careers in the industry.

## Key Things I Learned
- Offensive Security: Did a simple exercise hacking a vulnerable banking website to see basic exploitation in action.
- Defensive Security: How analysts detect and respond to attacks.
- Careers in Cyber: Roles like SOC Analyst, Penetration Tester, and GRC. I’m leaning towards SOC / blue team for entry-level opportunities.
- CIA Triad: Confidentiality, Integrity, and Availability — the core principles of security.

## Main Concepts
- Ethical hacking always requires permission
- Difference between threat, vulnerability, and risk
- Red team vs blue team approaches

## Reflections
This module helped me understand the bigger picture and confirmed I want to focus on defensive work first. It also showed me how much there is to learn.
